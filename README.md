# Bishop AI Dark Sky Plugin

This is a plugin for [Bishop AI](https://github.com/bishop-ai/bishop-ai).

Ask Bishop AI about the weather and get forecasts from [Dark Sky](https://darksky.net)

## Examples
- "What is the weather like?"
- "What is the temperature like tomorrow?"
